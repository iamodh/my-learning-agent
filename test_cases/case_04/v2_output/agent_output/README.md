# 문서 배치

이번 학습은 Redis 캐싱 개념 하나와, 그것을 외부 API 응답 캐싱에 적용한 결과 하나로 구성된다. 문서가 두 개뿐이므로 분류 폴더를 만들지 않고 평탄하게 둔다.

```
agent_output/
├── README.md                       이 파일
├── redis-캐싱.md                   지식 문서 (개념 + 하위 측면을 섹션으로 통합)
└── 외부-api-응답-캐싱-적용.md      결과 문서 (만들려던 것 + 참조 매핑)
```

## 인과 관계

- `redis-캐싱.md`에서 학습한 개념(메모리 기반 속도, 기본 패턴, TTL, 키 네이밍, 무효화, 스탬피드)이
- `외부-api-응답-캐싱-적용.md`의 외부 API 캐싱 구성 결정에 사용되었다.
