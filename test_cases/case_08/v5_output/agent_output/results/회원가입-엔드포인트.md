# 회원가입 엔드포인트

Express의 `/signup` 라우트에서 이메일/비밀번호를 받아 비밀번호를 bcrypt로 해싱한 뒤 사용자 레코드를 생성한다.

## 구현

```js
const bcrypt = require('bcrypt');
const saltRounds = 12;

app.post('/signup', async (req, res) => {
  const { email, password } = req.body;
  const hashed = await bcrypt.hash(password, saltRounds);
  await User.create({ email, password: hashed });
  res.json({ ok: true });
});
```

평문 비밀번호는 DB에 저장하지 않는다. `bcrypt.hash`가 salt를 자동으로 생성해 해시 문자열에 포함시키므로 salt를 별도 컬럼으로 다룰 필요가 없다.
