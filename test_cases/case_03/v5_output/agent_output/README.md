# Express JWT 인증 학습 정리

대화에서 Express 서버에 JWT 인증을 붙이려는 목표로 JWT의 본질과 액세스/리프레시 토큰 패턴을 학습하고, 로그인·리프레시 엔드포인트를 구현했다.

## 구조

```
.
├── README.md
├── knowledge/
│   └── jwt-auth.md          # JWT 개념 + jsonwebtoken 사용 + 액세스/리프레시 패턴
└── result/
    └── express-jwt-auth.md  # Express 로그인 + 리프레시 엔드포인트 구현
```

## 문서 관계

- `result/express-jwt-auth.md`는 `knowledge/jwt-auth.md`를 참조한다.
