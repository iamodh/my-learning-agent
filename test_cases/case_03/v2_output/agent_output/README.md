# 문서 배치

문서가 두 개뿐이라 분류 폴더 없이 평탄하게 둔다.

```
.
├── jwt.md                  # 학습한 지식: JWT 개념·구조·페이로드·액세스/리프레시 토큰·회전 방어
└── express-jwt-auth.md     # 결과: Express 서버에 JWT 인증을 붙인 산출물 + 참조 매핑
```

- `jwt.md`: JWT 한 개념을 중심으로 페이로드 원칙·서명 vs 암호화·액세스/리프레시 토큰 전략·회전 방어를 섹션으로 통합했다.
- `express-jwt-auth.md`: 만들려 했던 산출물(Express + JWT 인증)과 사용된 지식의 참조 이유.
