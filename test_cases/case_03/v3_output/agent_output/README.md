# 학습 정리: Express + JWT 인증

## 폴더 구조

```
.
├── README.md
├── knowledge/
│   ├── jwt.md
│   ├── access-refresh-token.md
│   └── refresh-token-storage.md
└── result/
    └── express-jwt-auth.md
```

- `knowledge/` — 인증 시스템을 설계하면서 학습한 개념들
- `result/` — 실제로 만들기로 한 Express JWT 인증 시스템
