# Next.js 개인 블로그 학습 정리

대화에서 학습한 Next.js 개인 블로그 구축 흐름을 지식 문서와 결과 문서로 정리한다.

## 폴더 구조

```
agent_output/
├── README.md
├── result/
│   └── nextjs-personal-blog.md
└── knowledge/
    ├── nextjs-app-routing-and-ssg.md
    ├── mdx-in-nextjs.md
    ├── vercel-deployment.md
    └── nextjs-seo-metadata.md
```

## 문서 안내

- `result/nextjs-personal-blog.md` — 개인 블로그 구축 계획과 사용된 지식 매핑.
- `knowledge/nextjs-app-routing-and-ssg.md` — app 디렉토리 라우팅, 동적 경로, generateStaticParams, ISR.
- `knowledge/mdx-in-nextjs.md` — MDX 포맷, 프론트매터, @next/mdx 설정, 전역 컴포넌트 등록.
- `knowledge/vercel-deployment.md` — Vercel 배포 흐름, 환경 변수 처리.
- `knowledge/nextjs-seo-metadata.md` — metadata API, Open Graph/Twitter, generateMetadata, sitemap/robots.
