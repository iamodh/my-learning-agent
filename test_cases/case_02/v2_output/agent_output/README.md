# 문서 구조

Express 이미지 업로드 API를 만들기 위해 학습한 지식과 그 결과물을 정리한다.

```
.
├── README.md
├── multer.md              # 지식: multer 미들웨어 (multipart 파싱, 스토리지, fileFilter, limits)
└── image-upload-api.md    # 결과: Express 이미지 업로드 API
```

- multipart/form-data·스토리지 어댑터·fileFilter·limits는 모두 multer를 학습하다 마주친 하위 측면이라 `multer.md` 한 문서의 섹션으로 통합했다.
- 지식·결과 각각 문서가 하나뿐이라 분류 폴더 없이 루트에 평탄하게 배치했다.
- `image-upload-api.md`는 사용된 지식과 참조 이유를 표로 매핑한다.
