# nodemailer

`nodemailer`는 Node.js에서 이메일을 보낼 때 사실상 표준으로 쓰이는 라이브러리다. 서비스에 이메일 발송 기능을 붙일 때 가장 먼저 떠올리는 선택지로 언급되었다.

## 설치

`npm install nodemailer`로 설치하면 바로 사용할 수 있다.

## 사용 흐름

1. SMTP 호스트, 포트, 계정 정보를 넣어 transporter를 만든다.
2. 만든 transporter에서 `sendMail`을 호출하면 메일이 발송된다.

대화에서는 이 두 단계까지만 다뤄졌다. 구체적인 옵션 키, 인증 방식의 세부, 에러 처리 흐름 등은 다루지 않았으므로 여기서는 더 깊이 들어가지 않는다.
