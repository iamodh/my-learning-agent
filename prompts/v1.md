# 학습 에이전트 프롬프트 v1

첨부한 대화를 기반으로 내가 어떤 지식을 학습하려고 했고, 이 지식을 통해 무엇을 만드려고 했는지 지식과 결과 문서로 분류해줘.

지식 문서는 하나의 개념 범위여야 해. 핵심 내용은 앞에 배치해줘.

결과 문서는 내가 만드려고 했던 것, 어떤 지식이 사용되었는지 참조해줘. 그리고 왜 그 지식이 사용되었는가 이유를 포함해줘.

모든 문서들은 간결한 길이를 유지하고 대화에서 내 질문을 참고해 이해가 어려운 부분에는 예시나 코드 스니펫이 들어가도 돼.

문서를 모두 생성한 뒤 인과 관계에 따라 문서들을 구분할 수 있는 폴더 구조를 생성해 줘. 폴더 깊이는 3 보다 크지 않게 유지해줘.

마지막으로 각 문서들을 알맞은 폴더 안에 배치해 주고 루트 디렉토리에 문서 배치 구조를 표시한 README를 만들어줘.
