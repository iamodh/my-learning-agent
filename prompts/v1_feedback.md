## 2. 결함 (영향 큰 순)

### D1. 지식 문서 과분할 — 최우선 (FAIL 6건 중 4건의 직접 원인)

**증상**: 하나의 개념으로 묶여야 할 지식이 여러 파일로 쪼개진다. 그 결과
지식 문서 수·총 문서 수가 부풀고, "하위 주제는 상위 문서의 섹션으로
통합"이라는 기대가 연쇄적으로 깨진다. 통과한 케이스(01·08)는 공통적으로
한 개념을 한 문서로 묶었고, 실패 케이스는 예외 없이 부수 주제를 독립
파일로 떼어냈다.

**원인**: v1.md의 `"지식 문서는 하나의 개념 범위여야 해"`는 *개념(독립
문서)* 과 *하위 측면(문서 내 섹션)* 을 가르는 기준을 주지 않는다.
에이전트는 모든 부수 주제를 독립 개념으로 보고 파일을 새로 만든다.

**v2 처방**:
- 분할 기준을 명문화한다. 예: *"부모 개념을 떼면 단독으로 의미가
  성립하지 않거나, 사용자가 그 부모 개념을 학습하다 마주친 세부 주제는
  새 파일이 아니라 `##` 섹션으로 통합한다. 새 지식 문서는 그 개념이
  서로 무관한 여러 결과에서 독립적으로 재사용될 때만 정당하다."*
- **문서 수 최소 편향**을 기본값으로 건다: "쪼갤지 합칠지 애매하면 합친다."
- 자가 점검 지시 추가: *"두 번째 이상의 관련 지식 파일을 만들기 전에,
  그것이 기존 문서의 섹션이 될 수 없는지 먼저 자문하라."*

### D2. 결과 문서 강제 생성 (case 05)

**증상**: 제작 의도가 약하거나 스치듯 언급된 입력인데도 결과 문서와
참조 매핑·참조 이유까지 만들어낸다.

**원인**: v1.md는 결과 문서를 *항상* 만드는 산출물로 전제한다. 생성
조건이 없다.

**v2 처방**: 결과 문서를 **조건부 산출물**로 바꾼다. *"명확한 제작
의도가 있을 때만 결과 문서를 만든다. 대화가 주로 지식 습득이고 만들려는
대상이 가정·여담 수준이면 지식 문서만 만들고 결과 문서·참조 매핑·참조
이유를 생성하지 않는다."*

### D3. 스코프 이탈 / 환각 보강 (case 05, 07)

**증상**: 대화에 없던 설정·옵션·대안 비교 문서를 추가하거나, 얕게
언급된 항목을 추측으로 상세화한다.

**v2 처방**: *"산출물은 대화에서 실제로 확립된 내용으로 엄격히
한정한다. 대화에서 다루지 않은 설정값·옵션·대안은 추가하지 않는다.
얕게 언급된 주제는 얕게 남긴다(추측성 보강 금지)."*

### D4. 탈락한 대안을 결과 문서가 참조 / 결정 번복 처리 (case 07; case 10은 잘 처리—대조군)

**증상**: 사용자가 선택을 번복했는데, 결과 문서가 *탈락한* 대안을 사용
지식으로 참조하고 결정 과정을 시간순으로 나열한다.

**v2 처방**: *"결과 문서는 **최종 결정**의 근거 지식만 참조한다.
탈락한 대안은 지식 문서로 존재할 수 있으나 결과 문서의 참조 매핑에는
넣지 않는다. 결정 번복은 한 줄 요약으로만 남기고 시간순으로 재현하지
않는다."*

### D5. 폴더 깊이·과분류 (case 06, 경미: 05)

**증상**: 파일 1~2개뿐인데 카테고리 하위 폴더를 만들어 불필요하게
깊어진다.

**v2 처방**: v1의 "깊이 ≤ 3"은 상한일 뿐이다. *"인과 관계를 표현하는
가장 얕은 구조를 택한다. 파일이 1~2개인 분류 폴더는 만들지 않는다."*

## 3. 회귀 방지 (v1이 잘한 것 — v2에서 깨지 말 것)

전 케이스에서 일관되게 통과한 항목: **참조 이유 명시**, **잡담의 본문
유입 차단**, **예시 요청 개념에 코드 스니펫 포함**, **지식 문서 첫 문단을
코드가 아닌 정의 텍스트로 시작**. D1~D5를 고치며 이 4개를 약화시키지
않는지 확인한다.

## 4. v2 작업 우선순위

1. **D1** (과분할 억제 규칙 + 통합 편향) — 단일 최대 레버리지, FAIL 다수를 PASS로 전환.
2. **D2 / D4** — 규칙 공백이 뚜렷한 단일 케이스 실패. 짧은 조건문으로 해결.
3. **D3** — 일반 가드레일.
4. **D5** — 경미, 한 문장 보강.

검증은 v2 작성 시 보지 않은 **홀드아웃 케이스**로 한다(같은 케이스로
튜닝+평가 시 과적합).
